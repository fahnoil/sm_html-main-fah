# SM-HTML-01

แอปพลิเคชันเว็บสำหรับจัดการข้อมูลด้านสุขภาพและการแพทย์ที่ครอบคลุม พร้อมฟีเจอร์สำหรับจัดการข้อมูลผู้ป่วย การนัดหมาย การจัดการระบบแจ้งเตือน และการแสดงข้อมูลทางคลินิก

## ภาพรวม

SM-HTML-01 เป็นแอปพลิเคชันเว็บด้านสุขภาพอย่างเต็มรูปแบบที่ออกแบบมาเพื่อจัดการข้อมูลผู้ป่วย ข้อมูลทางคลินิก การนัดหมาย และระบบแจ้งเตือนแบบเรียลไทม์ แอปพลิเคชันนี้มี Dashboard ที่ใช้งานง่าย พร้อมโมดูลเฉพาะสำหรับด้านต่างๆ ของการจัดการสุขภาพ

## การติดตั้ง

1. โคลนหรือดาวน์โหลดริโปสิทอรี่
2. วางโฟลเดอร์ `sm_html` ในไดเรกทอรี่รูทของเซิร์ฟเวอร์เว็บของคุณ (เช่น `/Applications/MAMP/htdocs/`)
3. ตรวจสอบว่าเซิร์ฟเวอร์เว็บ (Apache, Nginx ฯลฯ) ทำงานอยู่
4. เปิดเบราว์เซอร์และไปที่ `http://localhost/sm/sm_html/`

## วิธีใช้งาน

1. **เข้าถึงแอปพลิเคชัน**: เปิด `index.html` หรือไปที่หน้าเข้าสู่ระบบ
2. **การรับรองความถูกต้อง**: ใช้ข้อมูลประจำตัวของคุณเพื่อเข้าสู่ระบบบน `login.html`
3. **Dashboard**: เข้าถึง Dashboard หลักสำหรับภาพรวมและการนำทาง
4. **การจัดการข้อมูลผู้ป่วย**: ใช้หน้า Patient-Profile และ Patient-Registry สำหรับข้อมูลผู้ป่วย
5. **การสร้างรายงาน**: สร้างรายงานโดยใช้หน้าต่างๆ (rpt_01 ถึง rpt_04)

## ฟีเจอร์

- 🔐 **การรับรองความถูกต้องที่ปลอดภัย**: ระบบเข้าสู่ระบบสำหรับการควบคุมการเข้าถึงผู้ใช้
- 👥 **การจัดการข้อมูลผู้ป่วย**: โปรไฟล์ผู้ป่วยและทะเบียนที่ครอบคลุม
- 📋 **การจดบันทึกทางคลินิก**: แบบฟอร์มการคัดกรองและการป้อนข้อมูลทางคลินิก
- 📊 **Dashboard รายงาน**: รายงานหลายประเภทสำหรับการวิเคราะห์ข้อมูล
- 📱 **การแพทย์ระยะไกล**: ความสามารถในการให้ความเห็นทางการแพทย์ระยะไกล
- 🚨 **ระบบแจ้งเตือน**: การจัดการระบบแจ้งเตือนแบบเรียลไทม์
- 📍 **แผนที่ภูมิศาสตร์**: SpotMap สำหรับการแสดงภาพข้อมูลตามตำแหน่ง
- ♻️ **การประเมินความเสี่ยง**: การวิเคราะห์คุณภาพอากาศและความเสี่ยงต่อสุขภาพ
- 🔍 **การค้นหาขั้นสูง**: การค้นหา PDW สำหรับการค้นหาข้อมูลที่ซับซ้อน
- 📱 **การออกแบบแบบตอบสนอง**: เหมาะสำหรับการแสดงผลบนเดสก์ท็อปและแท็บเล็ต

## เทคโนโลยีที่ใช้

- **Frontend**: HTML5, CSS3
- **การออกแบบ**: ไฟล์ CSS แบบกำหนดเองสำหรับการออกแบบโมดูลาร์
- **เซิร์ฟเวอร์**: MAMP (Apache)
- **การปรับใช้**: การปรับใช้บนเซิร์ฟเวอร์หรือท้องถิ่น

## การจัดระเบียบไฟล์

### ไฟล์ CSS (`/css` folder)

แต่ละหน้า HTML มีไฟล์ CSS ที่สอดคล้องกันสำหรับการออกแบบ:

```
css/
├── style.css              # สไตล์ทั่วไป
├── login.css              # สไตล์หน้าเข้าสู่ระบบ
├── dashboard.css          # สไตล์ Dashboard
├── admin.css              # สไตล์แผงผู้ดูแลระบบ
├── Patient-Profile.css    # สไตล์โปรไฟล์ผู้ป่วย
├── Patient-Registry.css   # สไตล์ทะเบียนผู้ป่วย
├── Screening_CRF.css      # สไตล์แบบฟอร์มทางคลินิก
├── visit.css              # สไตล์บันทึกการเข้ารับการรักษา
├── Telehealt.css          # สไตล์การแพทย์ระยะไกล
├── AlertCenter.css        # สไตล์ระบบแจ้งเตือน
├── SpotMap.css            # สไตล์แผนที่ภูมิศาสตร์
├── airisk.css             # สไตล์การประเมินความเสี่ยง
├── PDW_Search.css         # สไตล์การค้นหาขั้นสูง
├── rpt_01.css ถึง rpt_04.css  # สไตล์รายงาน
└── [ไฟล์ CSS เฉพาะหน้าอื่นๆ]
```

## เริ่มต้นใช้งาน

### เริ่มต้นอย่างรวดเร็ว

1. เริ่มต้นเซิร์ฟเวอร์ MAMP
2. ไปที่ `http://localhost/sm/sm_html/login.html`
3. ป้อนข้อมูลประจำตัวของคุณ
4. สำรวจ Dashboard และฟีเจอร์ต่างๆ

### การพัฒนา

เพื่อแก้ไขหน้าและสไตล์:
1. แก้ไขไฟล์ HTML ที่สอดคล้องกัน
2. อัปเดตไฟล์ CSS ที่เชื่อมโยงในโฟลเดอร์ `/css`
3. รีเฟรชเบราว์เซอร์เพื่อดูการเปลี่ยนแปลง

## หมายเหตุ

- แต่ละหน้ามีไฟล์ CSS ที่สม่ำเสมอเพื่อการเดินทางง่ายๆ
- ทรัพยากรคงที่ทั้งหมดจัดระเบียบในไดเรกทอรี่รูทและ CSS
- แอปพลิเคชันออกแบบมาสำหรับเบราว์เซอร์เว็บสมัยใหม่ (Chrome, Firefox, Safari, Edge)
- ตรวจสอบว่า MAMP หรือเซิร์ฟเวอร์ที่คล้ายกันทำงานอยู่เพื่อการทำงานที่ถูกต้อง
